Test Plan: Банковское приложение v1.0

## 1. Введение
Объект тестирования: Консольное банковское приложение на Python  
Версия: 1.0  
Цель тестирования: Проверить базовую функциональность регистрации, авторизации и управления балансом

## 2. Объем тестирования
Включено:

- Авторизация в системе

- Обработка ошибок ввода


## 3. Подход к тестированию
- Функциональное тестирование - проверка основной функциональности
- Негативное тестирование - проверка обработки ошибок
- Usability тестирование - удобство использования

## 4. Критерии начала/окончания
Начало тестирования: Все функции v1.0 реализованы
