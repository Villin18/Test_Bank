# Тестовое банковское приложение

Консольное банковское приложение на Python с тестированием

## Функциональность v1.0
-  Регистрация пользователей
-  Авторизация в системе  
-  Просмотр баланса
-  Выход из системы

## Структура проекта
-  main.py Исходный код
-  test_plan.md тест план для проекта
-  test_cases.md тест кейсы по проекту
-  bug_reports.md файл с найденными багами
