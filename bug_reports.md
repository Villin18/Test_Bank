# Баг-репорты

## BUG-REG-001: Отсутствует валидация пустых полей при регистрации
 
**Серьезность**: High  
**Приоритет**: High  
**Модуль**: Регистрация  
**Окружение**: 
- OS: Windows
- Python: Python 3.14.0
- Версия приложения: 1.0

**Статус**: ✅Fixed  

### Описание
При попытке регистрации с пустыми полями система не выполняет проверку на заполненность обязательных полей и позволяет создать пользователя с пустыми данными

### Шаги воспроизведения
1. Запустить приложение
2. Выбрать "1. Регистрация"
3. На вопрос "Как вас зовут:" нажать Enter (оставить пустым)
4. На вопрос "Укажите ваш логин:" нажать Enter (оставить пустым) 
5. На вопрос "Введите пароль:" нажать Enter (оставить пустым)
6. Наблюдать результат

### Фактический результат
- Регистрация "успешно" завершается
- В базе данных создается пользователь с пустыми полями name, login, password
- Сообщение: "Регистрация успешна!"

### Ожидаемый результат
- Валидационная ошибка при пустых полях
- Сообщение: "Ошибка: Все поля обязательны для заполнения!"
- Данные пользователя не сохраняются в БД
## Описание
- Добавилась проверка на наличие пустых полей
